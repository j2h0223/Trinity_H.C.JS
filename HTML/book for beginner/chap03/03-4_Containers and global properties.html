전역 속성(Global attribute)

모든 html 태그에서 공통적으로 사용할 수 있는 속성
사용 빈도가 높은 대표적인 전역 속성
id : 요소에 고유한 이름을 부여하는 식별자 역할
class : 요소를 그룹 별로 묶을 수 있는 식별자 역할
title : 요소에 대한 추가 정보를 기입할 수 있는 텍스트 속성, 사용자에게 툴팁 제공

고유한 식별자, id

스크립트 및 스타일 적용 시 특정 요소를 식별하기 위해 요소에 대한 고유식별자를 지정
각 요소에 대한 id는 고유한 값
